# dotfiles

# Dependencies

+ **Qtile 0.31.0**: Window manager
+ **Qtile 1.7.5**: Window manager
+ **Picom v12**: Compositor
+ **pactl 16.1**: Volume control
+ **playerctl v2.4.1**: Control currently playing media
+ **alacritty 0.16.0-dev**: Terminal emulator
+ **fish version 4.0.2**: Shell
+ **fisher version 4.4.5**: Package manager for fish
+ **tide version 6.1.1**: Fish prompt
+ **pyxdg 0.28 (python)**: Required for Qtile's StatusNotifier widget
+ **dbus-fast 2.39.3 (python)** (or dbus-next): Required for Qtile's StatusNotifier widget
+ **dbus-next 0.2.3 (python)**: Required for Qtile's Mpris2 widget
+ **psutil 5.9.8 (python)**: Required for Qtile's CPU widget
+ **xfsettingsd 4.18.4**: XFCE's settings daemon (convenient to have)
+ **dunst 1.9.2**: Notification daemon
+ **XFCE4 Power Manager 4.18.3**: Power manager from the XFCE DE
+ **Neovim v0.12.0-dev**
+ **Yazi 25.6.11**
